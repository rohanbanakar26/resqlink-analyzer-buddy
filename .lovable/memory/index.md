# Project Memory

## Core
ResQLink: Emergency response network — citizens report, NGOs coordinate, volunteers respond.
Emergency red primary (#ef4444). Inter font. Mobile-first with bottom nav.
Lovable Cloud backend with real auth, DB persistence, and realtime. 3 roles: citizen, volunteer, NGO.
Categories: food, disaster, medical, shelter, senior, education, sanitation.
Priority scoring: urgency*40 + totalReports*30 + recentReports*20 + severity*10.
Auto-confirm email enabled for easier testing.
